
# 🔐 Password Strength Checker

A simple Python-based cybersecurity project that evaluates the strength of a password using basic security principles.

---

## 🚀 Features
- Checks password length
- Detects uppercase letters
- Detects numbers
- Detects special characters (symbols)
- Classifies passwords as:
  - Weak
  - Medium
  - Strong

---

## 🧠 Concepts Used
- String handling
- Conditional logic
- Boolean checks
- Cybersecurity fundamentals (password validation)

```bash

```

2. Navigate into the folder:
```bash
cd password-strength-checker
```

3. Run the program:
